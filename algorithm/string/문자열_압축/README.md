# 문자열 압축

<h3>설명</h3>

알파벳 대문자로 이루어진 문자열을 입력받아 같은 문자가 연속으로 반복되는 경우 반복되는

문자 바로 오른쪽에 반복 횟수를 표기하는 방법으로 문자열을 압축하는 프로그램을 작성하시오.

단 반복횟수가 1인 경우 생략합니다.

<h3>입력</h3>

첫 줄에 문자열이 주어진다. 문자열의 길이는 100을 넘지 않는다.

<h3>출력</h3>

첫 줄에 압축된 문자열을 출력한다.

<h3>예시 입력 1</h3>

KKHSSSSSSSE

<h3>예시 출력 1</h3>

K2HS7E

<h3>예시 입력 1</h3>

KSTTTSEEKFKKKDJJGG

<h3>예시 출력 1</h3>

KST3SE2KFK3DJ2G2

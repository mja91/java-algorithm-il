# 최대점수 구하기(DFS)

<h3>설명</h3>

1부터 N까지 번호가 적힌 구슬이 있습니다.

이 중 중복을 허락하여 M번을 뽑아 일렬로 나열하는 방법을 모두 출력합니다.

<h3>입력</h3>

첫 번째 줄에 자연수 N(3<=N<=10)과 M(2<=M<=N)이 주어집니다.

<h3>출력</h3>

첫 번째 줄에 결과를 출력합니다.

출력순서는 사전순으로 오르차순으로 출력합니다.

<h3>예시 입력 1</h3>

3 2

<h3>예시 출력 1</h3>

1 1<br>
1 2<br>
1 3<br>
2 1<br>
2 2<br>
2 3<br>
3 1<br>
3 2<br>
3 3
